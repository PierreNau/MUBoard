INSTALLATION INSTRUCTIONS
-------------------------
1) Copy MUBoard to your modules folder.
2) You will find the folder 'userdata/MUBoard' in the folder Resources. 
   Copy this one with all subfolders into the folder 'userdata' or your personal data folder ( setting in the config.php ) of your zikula installation and give it and all subfolders permissions 0777.
3) Install and enable the module in the extension manager
